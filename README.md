## ЁЯЪА Getting Started

### Prerequisites
- Node.js
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Galaxy-Vortex/Glint
   ```

2. Install dependencies:
   ```bash
   npm i
   ```

3. Start the server:
   ```bash
   npm start
   ```
